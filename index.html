Bond Scenario Simulator

Estimate how bond prices react to interest rate shocks using modified duration and convexity. Built for investors, macro watchers, and anyone who wants a cleaner way to think about rate sensitivity.

This tool is designed for scenario analysis, not portfolio advice. It helps translate rate moves into estimated bond price impact using standard fixed-income approximations.
